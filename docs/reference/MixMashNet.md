# MixMashNet: Multilayer and Single-Layer Network Modeling

Tools for estimating single-layer and multilayer Mixed Graphical Models,
bootstrap-based membership stability, centrality/bridge metrics, and
plotting. Includes masked estimation for constrained (e.g., multilayer)
structures and safe graph assembly adapted from mgm.
